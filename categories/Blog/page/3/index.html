编码风格的设定

之前一直没注意过，因为和人协作开发，发现存在编码排版格式不统一的问题。

我是用emacs，tab是四个空格，这个没问题，但是缩进就不对了，而另一个人是用Zend Studio，tab和缩进都是四个空格。怎么个意思呢，而且发现有时候的缩进非常奇怪:

if (true)
    {
    }

花了点时间想解决缩进的问题，结果只是一个风格设置的问题。。

几种编码风格在这里, 设置的话用C-c . 或者M-x c-set-style.参考这里 .

更进一步自定义:

(c-add-style “immediate”
‘(“K&R”
(c-basic-offset . 4)))
(setq c-default-style “immediate”)

从org2blog开始使用org-mode

貌似用Emacs的人中不用org-mode的不多，也听说过一些人是因为org-mode才开始接触emacs的，org-mode很强大我也知道，可是不知道怎么了，我就是没怎么用org。这不，现在开始想好好学学嘛。

现在用的是org2blog,用来发博客的，因为只能发，不能从WP上将日志获取下来，找了个脚本wp_to_org2blog.py, 这个我改了下，之前的只能将所有日志导出到一个文件，现在可以按’year-month’来建目录，然后以标题来做文件名保存。

目前遇到的问题是 (1)行首空格搞不定 (2)换行设置了没用，手动启用有效 (3)能否直接按标题来保存文件。

暂时先这样吧，这些问题应该随着对org-mode的了解增多会慢慢解决的。

感恩节

今晚搞定了AC。
auto-complete clang 有一个

  (setq ac-clang-auto-save t)

的设置，而这个是在clang中为了自动补全而有意为之的，但是对于非c的码农来说这个就有点二了，因为这个设置会让你在触发了ac而有没有或已经补全完成了时自动将当前buffer保存，这很不和谐。
将ac的弹出窗口上下选择绑定到了合适的位置

(define-key ac-menu-map “C-n” ‘ac-next)
(define-key ac-menu-map “C-p” ‘ac-previous)

这样不会不顺手了。
找了份yasnippet的php-mode snippets，路径也重新设置了，之前yasnippet自带的snippets目录不用了。

这是我的配置(https://github.com/xlmo/prelude),是在prelude的基础上改的，顺便说一句，Emacs prelude真的很棒。

还有一个RO港中。

感觉最吭爹的是那个网络，我擦。

  • 第一次使用Near，需要开VPN改DNS才能连上。之后就不用开VPN了。

升级Mountain Lion

花了一通宵挂机下载了Mountain Lion，花了不到三十分钟升级系统。

  1. 没了web共享。
  2. 没了X。一些软件用不了，例如Cinch。（装xquartz 即可）
  3. 软件更新整合到了Mac App Store里面。
  4. 新的中文输入法。一样的不好用。
  5. 因为通知中心的存在，可以卸载Growl了。设置方式和iOS上的很像，没有学习成本。
  6. 增加了听写功能。但是目前好像不支持中文。累死Siri，也是把语音发送给Apple，然后返回识别出的结果。
  7. 新增Game Center。基本上对我没用。
  8. 信息。之前Lion上有Beta的“信息”应用，现在正式版支持iMessage了。
  9. 提醒事项。和iOS上的差不多。
  10. 备忘录。这是把之前Mail里的备忘录单独拿出来做了一个App。
  11. 分享。支持新浪微博、优酷、土豆、flickr、twitter的系统级分享。
  12. 其他。相应的iPhoto/iMovie/Page/Keynote/Numbers都升级以支持Mountain Lion，并加入了iCloud支持。同时Xcode也更新到4.4了。
  13. 图标细节有改变。例如废纸篓变得更高档了…… Dock App下的小圆点变成小横条了。
    值得注意的是，10.8中Apple将Web 共享这个选项从共享设置中移除了，但是apache和php却没有一块移除，所以只要做一些小修改就能跑起来:
  14. /etc/apache2/httpd.conf 取消注释加载libphp的那一行。
  15. 在/etc/apache2/users/目录下建立xx.conf,xx是你用户名。

  16. 编辑内容


    Options -Indexes MultiViews
    AllowOverride All
    Order allow,deny
    Allow from all

重启apache。

近况

早几天终于将我唯一的一个备案域名给绑定到了SAE。流程还算顺利，时间还算给力，访问速度没得说。
基本上，如果不是之前将应用绑定到了海外的话，一天就可以办下来。因为要取消绑定，再重新绑定，耽误了一天。就这样也两天搞定，确切的说是两个工作日搞定。

抽空看完了Objectvie C 和Cocoa 两本书，正在看iOS编程。为什么要学iOS/Mac开发呢？因为心里有一个应用的设想，一个雏形，能满足我的一个需求，打算慢慢实现这个设想。

之前看个youku，机器声音巨大，拆机给风扇清理了一下，立马安静了。
之前换SSD的时候本来是打算把光驱卸下来换硬盘上去的，连光驱位插槽都买好了，结果工具不合适，螺丝有拧花的迹象，立马收手。乘着清理风扇的机会又对光驱下手了，
结果在我不屑的努力下，光驱位的两颗螺丝已经彻底磨平了。。。 至此，只能彻底打消这个主意了。

现在用着sublime text 2 感觉还不错，虽说是收费的，但是不注册除了偶尔保存的时候提示一下要注册以外，其他都是全功能使用，基本不影响正常使用，作者还是很厚道的。

说到厚道，就不得不提之前上架的Tiny Wings 2了。品质精良，趣味性和游戏性都很不错，更新的内容也挺丰富，而且并不是单独做为一个新的APP来卖，而是直接在1.0上升级而来，不用二次购买，也没有恶心的IAP，非常赞。

iOS上装了百度输入法3.0，目前用着挺满意。不得不提的是果断的用Chrome for iOS替换Safari成为默认浏览器了，Chrome的标签同步太好用了。

换了家SSH和手机VPN。
Evernote也付费了。
口水Macbook Pro 15 Retina Display。

完美越狱终于来了

今天iPhone 4S/iPad 2完美越狱出来了。真正的一键越狱，很快速很傻瓜。
装了如下软件来加强使用：

  • MultiIconMover 可以一次移动多个图标
  • GFWinterceptor 翻墙用
  • FolderEnhancer 一个文件夹可放入N多程序，打开文件夹也快了一些
  • KuaiDial 主要用到的就是一个黑名单和一个去电接通震动
  • Activator 少用Home键，用手势替代
  • iFile 必备
  • 百度输入法 搜狗输入法 这个感觉比百度的好用。九宫格输入法，越狱主要是为了这个
  • SBSETTING 用到了在通知中心快速开关WIFI和BT的功能
  • iGotYa 输错两次解锁码能拍下照片并将定位信息和照片发送到指定邮箱和手机
  • GuizmOVPN open vpn 客户端
