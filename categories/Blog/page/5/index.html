不管喜不喜欢，生活过的地方就是那样

让人窒息的熟悉感觉，穿越十年，丝毫不模糊。

还是那样的楼，还是那样的地，还是那样的景，可看它的心情却大不一样。让我唏嘘的不是岁月，而是对于猛回头发现诸事皆变的突然。

以前的人，还是在那，位置不同了。老师、同学、领导、路人，没有一个缺席，都在装扮着我的回忆。

校友，同学，我们何时荣归故里，我们何时举杯言欢，我们又何时形同陌路。

生活过的地方，因为有人，所以有事情，因为有事情，而产生经历，因经历而产生感情，因感情而牢记岁月。匆匆而过的只是时间，我们的心不会随时间的流逝而消逝，只会在灰蒙蒙的来路上，感受着熟悉的气息。

不管喜不喜欢，生活过的地方就是那样，总是会在你毫无防备的时候让你颤抖，因为真的忘不了。

Centos环境配置

网络安装，地址填写
mirrors.163.com
/centos/5.6/os/i386/

修改时区和同步时间

yum install ntp
cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
ntpdate us.pool.ntp.org

安装Git及其依赖包

yum install zlib-devel openssl-devel perl cpio expat-devel gettext-devel
wget -c http://kernel.org/pub/software/scm/git/git-1.7.4.5.tar.bz2
tar jxvf git-1.7.4.5
autoconf
./configure —with-curl=/usr/local
make
make install

安装mysql

yum install mysql mysql-server

设置为启动项

chkconfig —levels 235 mysqld on

启动

/etc/init.d/mysqld start

设置密码

mysqladmin -u root password 密码

安装apache

yum install httpd

设置为启动项

chkconfig —levels 235 httpd on

启动

/etc/init.d/httpd start

安装php 5.3及扩展

yum install php53 php53-mysql php53-gd php53-imap php53-ldap php53-xml php53-xmlrpc php53-mbstring

配置防火墙

setup Firewall设置允许www服务

第三方库

wget -c http://packages.sw.be/rpmforge-release/rpmforge-release-0.5.2-2.el5.rf.i386.rpm
rpm -ivh rpmforge-release-0.5.2-2.el5.rf.i386.rpm

memcached

yum install memcached —skip-broken

启动Apache时，发现了一例错误：

Starting httpd: Warning: DocumentRoot [/www/mysite] does not exist

查询资料，在国外论坛上找到了答案：

“Turned out to be SELINUX.
Edit /etc/sysconfig/selinux and change it to disabled then reboot.”

即：

vi /etc/sysconfig/selinux

设置

SELINUX=disabled

然后

reboot

重启系统即可

小组级GIT服务器搭建

在本地生成SSH公钥：

localhost:xx xlmo$ ssh-keygen -t rsa
Generating public/private rsa key pair.
Enter file in which to save the key (/Users/xlmo/.ssh/id_rsa): 
Enter passphrase (empty for no passphrase): 
Enter same passphrase again: 
Your identification has been saved in /Users/xlmo/.ssh/id_rsa.
Your public key has been saved in /Users/xlmo/.ssh/id_rsa.pub.
The key fingerprint is:
xxxxx....xxxxx
The key's randomart image is:
+--[ RSA 2048]----+
|   .o.           |
|  .  . .         |
| .    o          |
|. . .  .         |
| . + . oS.       |
|  o = + =        |
| o o B *         |
|  + o + o        |
| .    Eo         |
+-----------------+
传送key 
scp ~/.ssh/id_rsa.pub git@服务器:/tmp/
**登录服务器操作** 新建用户 
useradd git
#注意usesradd添加的用户暂时是被封锁的，也就是说无法登录，需要设置一下密码
passwd git
以后ssh登录免输入密码 
cat /tmp/id_rsa.pub >> authorized_keys
新建版本库并初始化 
mkdir /home/git/git
cd /home/git/git/
mkdir example.git
cd example.git
git --bare init
本地使用新服务器版本库 
git remote add origin git@服务器:example.git
git push origin master
若有问题可尝试修改下权限 
chmod 755 /home/git
chmod 700 /home/git/.ssh
chmod 644 /home/git/.ssh/authorized_keys
附:Centos下安装Git 
wget -c http://kernel.org/pub/software/scm/git/git-1.7.4.5.tar.bz2
tar jxvf git-1.7.4.5.tar.bz2
cd git-1.7.4.5
./configure --with-curl=/usr/local
make
make install

备份环境＋SSD

备份环境：
磁盘阵列硬盘盒使用http://item.taobao.com/item.htm?id=9326270331 368+30运费
硬盘使用希捷 1.5T 7200转32M SATAII 台式机硬盘 http://www.360buy.com/product/136659.html 509×2

SSD：
Intel G3 160 G SSD http://item.taobao.com/item.htm?id=3710525456 2320＋ 12 运费

光驱位托架 http://item.taobao.com/item.htm?id=7016257823 149

使用TimeMachine备份。

光驱位放置SSD，主盘位放置HDD不变，这样是因为只有主盘位支持SMS，而SSD不需要防震。

因没有合适的工具，卸光驱的螺丝快要被拧花了，所以果断放弃了，现在直接主盘位放SSD，光驱不动，外接硬盘放大数据文件，从SSD创建软链接到外接硬盘上。

相关优化：

将休眠模式改为只保存在内存中，这样可以节约8G的磁盘空间（这个大小和内存大小相同）

sudo pmset -a hibernatemode 0
sudo rm /var/vm/sleepimage

关闭访问时间记录(atime)：

在 /Library/LaunchDaemon 目录下创建一个文件，例如名为com.apple.hfs.noatime.plist，然后编辑这个文件内容为

<?xml version=”1.0” encoding=”UTF-8”?>
<!DOCTYPE plist PUBLIC “-//Apple//DTD PLIST 1.0//EN”
        “http://www.apple.com/DTDs/PropertyList-1.0.dtd"&gt;
<plist version=”1.0”>
    <dict>
        <key>Label</key>
        <string>com.apple.hfs.noatime</string>
        <key>ProgramArguments</key>
        <array>
            <string>mount</string>
            <string>-vuwo</string>
            <string>noatime</string>
            <string>/</string>
        </array>
        <key>RunAtLoad</key>
        <true />
    </dict>
</plist>

然后需要将这个文件的所有权设为系统 root

sudo chown root:wheel /Library/LaunchDaemons/com.apple.hfs.noatime.plist

查看是否生效：

mount

属性中查看是否有noatime

因是放置于主盘位，所以关闭SMS（自由落体保护）。

sudo pmset -a sms 0

Snow Leopard下使用trim enabler来打开非官方SSD的TRIM。

效果：

开机小菊花转半圈，登陆后瞬间到桌面……
