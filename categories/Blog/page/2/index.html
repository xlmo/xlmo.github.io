得空回长

周末回长沙了，以后如无特殊情况，尽量买软卧，硬卧人太多、太吵了。上车的、下车的、找座的、嗑瓜子的、聊天的、碎碎念的，太烦人了。还好回去的票是Z字头。

回来当天晚上找TZY吃饭，吃了跳跳吓、烤羊排什么的。话说我对长沙的美食，已经完全不懂了。。

吃完去洗脚，来了个中式足疗。跟泰式按摩一样让我痛不欲生……

然后又是宵夜，螃蟹什么的，味道很不错。

隔天表哥结婚，中午喜宴。

下午逛了定王台，没有买书的冲动，纯粹为了找找当年的感觉。教辅书占了很大一部分，很变态。

2013泰国游

4.2-4.8，曼谷、大城、芭提雅。

下飞机就换了truemove的卡，选的是599 1G流量的3G套餐，时间好像是三十天的。不论是速度还是信号，都比国内3G要好太多太多，而且没有GFW的存在，基本上打开网站和图片都是秒开。 而且人家虽说是1G流量，但是用完之后并不是断网，而是速度下降而已，还是可以继续用。这点又完爆国内运营商。

路边摊永远都是精华。曼谷也不例外。酒店门口的路边摊，价格便宜，味道很不错，两个人吃得饱饱的才100泰铢，合人民币才20来块钱。

人民很友好，很nice。不动泰语没关系，英语不好也没关系，基本上单词+肢体动作就能简单沟通了。

因为性产业合法，所以很多成人表演和洗浴按摩的。因为合法，所以守规矩，卫生、专业。

尝试了正宗的传统泰式按摩，200泰铢一小时，很便宜了。可能是因为宅的关系，没有怎么运动，所以做这个按摩感觉很痛苦……

芭提雅是男人的天堂，也是女人的天堂。男女的性工作者都有，当然，不男不女的也有。这里的微信很普及，搜附近的人大部分是性工作者，随便聊两句就能上门。

海鲜什么的很便宜，很新鲜，很美味。

去了真的就不想回来了。

卡西欧换代

之前用的卡西欧十年系列，从2006年用到了2013年初，确实很强，没换过电池。

新年了，不知道哪根筋不对劲，想换表了。看了一圈，偏爱电子表，最后选了G-SHOCK 5600J.

坛子里说这是性价比最高的一款，有电波，有太阳能，自动背光，又是经典造型，感觉很拉风，淘宝上RMB不到600块。

这样，还不犹豫，入手了。

电波接收的是日本波，差不多十多分钟才接收到，这到影响不大。目前来说，很满意。

唯一的一点点瑕疵：时间和日期不能默认在一屏显示，要看日期只能切换一下。

入手iPod Classic

想买iPod的原由说起来似乎不相干：

  • iOS 6上将Podcast和Musci拆分成了两个 App，对于我这种两者都要听，并且经常是交叉着听的人来说操作起来很麻烦。 为手机省电。 现在手机用来电话、短信、游戏、Apps，电池已经很吃力了少个功能，其他功能可以多用会。 防盗-_-。 不用手机上插着耳机线，可以少点被贼注意，话说我上一部iPhone 4就是这么被偷的。 至于iPod则拿手上，和Kindle放一起，应无意外。
    对于第一点，我有个这样的情景需求：出门在路上听Podcast，上地铁用Kindle看书，就用Music播放班得瑞之类的做背景音乐，下地铁又切换回Podcast…… 所以我需要做的操作是:

  1. 打开Music2. 选歌，播放3. 暂停，退回桌面，打开Podcast4. 选择，播放5. 暂停，切换到Music6. 播放……

    若在Podcast退回桌面后一段时间再用耳机控制播放的话，有时候居然播放的是Music……

    所以我不能忍了，需要一个单独的设备来播放音乐，那为什么是iPod Classic呢？ 很简单，它够大…… 我懒得换歌，就一次性把所有专辑全都导进去，够听几年的。。

    对了，我是听无损的，其实也不是，是Apple Lossless格式的，还行，比WAV差点，比MP3强许多，这样一首歌大概在20-30M左右，现在已经塞进去3000多首了，还剩60来G空间，很不错。。 资源网上找的，这里 无损音乐 这个我挂了一个礼拜才下完，当然是挑着下的……没有新专辑，这点比较遗憾。。

    其实说了这些原因，条条都经不起推敲…… 归根到底就一个原因： 长草了 。。

Emacs里使用词典

在Emacs里调用系统的Dictionary.app,网上找了下，有这么几种实现方式：

  1. 使用命令行”open dict:/// + word” 来调用词典APP 链接
  2. 使用单独的python脚本dict.py来返回查询结果，单开buffer来显示结果。我使用这个方法。
    dict.py
    #!/usr/bin/python

import sys
from DictionaryServices import *

def main():
word = sys.argv[1].decode(‘utf-8’)
result = DCSCopyTextDefinition(None, word, (0, len(word)))
print result.encode(‘utf-8’)

if name == ‘main‘:
main()
code:

(defun dictionary ()
 “dictionary.app”
 (interactive)

(let ((editable (not buffer-read-only))
(pt (save-excursion (mouse-set-point last-nonmenu-event)))
beg end)

(if (and mark-active
(<= (region-beginning) pt) (<= pt (region-end)) )
(setq beg (region-beginning)
end (region-end))
(save-excursion
(goto-char pt)
(setq end (progn (forward-word) (point)))
(setq beg (progn (backward-word) (point)))
))

(setq word (buffer-substring-no-properties beg end))
(let ((win (selected-window))
(tmpbuf “ dict-process “))
(pop-to-buffer tmpbuf)
(erase-buffer)
(insert word “n”)
(start-process “dict-process” tmpbuf “dict.py” word)
(select-window win)
)
))
(define-key global-map (kbd “C-c w”) ‘dictionary)
