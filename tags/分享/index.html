我用的Mac软件

常用软件,在Lion下工作良好。不包括CLI程序。

iTunes 必备

Xcode 必备

IMKQIM 必备，中文输入法

Speed Download 5 下载工具

Paragon NTFS for Mac OS X 读写ntfs分区工具

Zend Studio 写PHP用的

Xmarks for Safari 用来同步chrome和safari的书签

Twitter 官方客户端，比较好用

Tunnelblick 必备，OpenVPN

Transmit 必备，用过许多ftp客户端，感觉还是这个最好用。

TotalFinder 必备，标签化Finder

TextWrangler 取代记事本的工具

TextMate 因为中文及编码问题，基本上只用到了Run，当作一个调试工具了

SplayerX 视频就靠它了，不完美，但是比较完美的了。。

SplashtopRemoteStreamer 挂机下载用。

Snippets 代码片段，可通过MobileMe同步

Simple Comic 看漫画的工具

Sequel Pro MySQL客户端，居然是免费的。。

Remote Desktop Connection 连接Windows用的。

RAR Expander 解压rar

QQ 鸡肋

Parallels Desktop 虚拟机

Pages 必备

OmniFocus GTD工具，很好用

Numbers 必备

MySQLWorkbench 因为有了Sequel Pro,所以只用来做Data Modeling

mfTuneKit 用来修改休眠模式和一些乱七八糟的选项

MacUpdate Desktop 可以管理Mac App Store没有覆盖到的软件更新

Keynote 必备

iTerm 2 替代默认终端工具

iStat Menus 锦上添花的工具，显示系统信息，还是很方便的

iPhoto 必备

iDocument Lite 管理PDF文档

iChm 必备，看chm的

Google Chrome 用不惯Safari，还是喜欢用chrome

Firefox 调试专用

Go2Shell 可在终端中直达Finder当前目录

Evernote 必备

Dropbox 必备

Cinch 必备 最大化工具，功能单一，非常好用

Burn 刻录软件

Breeze 可以记录各应用位置，适合多显示器和大尺寸显示器使用

BetterTouchTool Magic Mouse 设置用，也可以配置Touch Pad

BackOff 关闭或开启iTunes的备份功能

AppCleaner 删除软件的软件

Alfred 必备，比spotlight好用N倍

Air Video Server 偶然用用，在iPad上看高清电影。

Adium IM，非常不错

A Better Finder Rename 8 批量重命名工具

1Password 必备，用来管理各种账号、密码、注册码和一些乱七八糟的东西，有Safari和chrome插件
