Centos环境配置

网络安装，地址填写
mirrors.163.com
/centos/5.6/os/i386/

修改时区和同步时间

yum install ntp
cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
ntpdate us.pool.ntp.org

安装Git及其依赖包

yum install zlib-devel openssl-devel perl cpio expat-devel gettext-devel
wget -c http://kernel.org/pub/software/scm/git/git-1.7.4.5.tar.bz2
tar jxvf git-1.7.4.5
autoconf
./configure —with-curl=/usr/local
make
make install

安装mysql

yum install mysql mysql-server

设置为启动项

chkconfig —levels 235 mysqld on

启动

/etc/init.d/mysqld start

设置密码

mysqladmin -u root password 密码

安装apache

yum install httpd

设置为启动项

chkconfig —levels 235 httpd on

启动

/etc/init.d/httpd start

安装php 5.3及扩展

yum install php53 php53-mysql php53-gd php53-imap php53-ldap php53-xml php53-xmlrpc php53-mbstring

配置防火墙

setup Firewall设置允许www服务

第三方库

wget -c http://packages.sw.be/rpmforge-release/rpmforge-release-0.5.2-2.el5.rf.i386.rpm
rpm -ivh rpmforge-release-0.5.2-2.el5.rf.i386.rpm

memcached

yum install memcached —skip-broken

启动Apache时，发现了一例错误：

Starting httpd: Warning: DocumentRoot [/www/mysite] does not exist

查询资料，在国外论坛上找到了答案：

“Turned out to be SELINUX.
Edit /etc/sysconfig/selinux and change it to disabled then reboot.”

即：

vi /etc/sysconfig/selinux

设置

SELINUX=disabled

然后

reboot

重启系统即可
