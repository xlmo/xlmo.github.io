写在今天

这两年来，发现我喝酒的频率和量是越来越多了。总是莫名其妙的就在喝酒了，有时是兴之所至，有时是别人劝的，也有时是被强迫的，不管怎么说，对喝酒貌似比以前更加抗拒了。因为真不知道为什么要喝酒。

这几年来，关心的事情少了，操心的事情也少了，而兴趣却多了一些。有人说我宅，我无奈的想，我不宅我干嘛啊？

我们总在说梦想，但是呢，我们却有总是在找借口。梦想太多，而理想太少。

这几年来，已经没有了看书的兴致了。总是用时间来做借口，其实心里很明白，是因为心态变了。

早几天哈7看完了，完结了，有点失落了。想想家里的那7本书，想想那年暑假，真觉得很恍惚，一不留神就过去几年了。

有些事看多了会麻木，但是态度是不会变的。

我看这天啊，是要变变了。

创业总结之一

经过这几个月的工作，项目终于上线了。通过这几天的数据来看，还是很不错的，每天的量都在上升，客户越来越多，用户也在稳步增长中，新服务器也到位了，虽然还有很多问题需要解决，但到现在，也算是一个阶段的完结了吧。

第一次创业，虽然还不算创业成功，但到目前为止，还是发现了很多问题，也许是通病吧，但确实是很要命的问题。

 

职责不明

作为一个创业团队来说，初期肯定会存在人手不足的问题。要么招人，要么每个人多兼顾一下。招人势必成本上升，管理会受到挑战；更多的团队会选择一人分饰两角或多角。这样就会存在一个职能划分的问题，比如客户投诉，在没有客服专员的情况下是由谁来处理？如果你的回答是谁有时间就谁处理，那么现实肯定是没有人有时间来处理这个用户的投诉。要知道，谁都能管，意味着没人会去管。没有专门的职责分配，在大家都处于饱和状态的情况下是没有人愿意去主动承担的。若职责不明，则会造成团队内部的推、托、赖。

规划没有

不管是创业团队还是大公司，对于每个阶段都需要计划或规划来指导。这里所说的规划，不是指今天干什么、明天干什么这种具体事务的工作计划，而是指团队的创业项目在执行上的一个基本目标。团队行进时可以走左边的弯路也可以走右边的捷径，但要有规划来指引，至少方向不能走错了。在创业初期，面对持续的成本增加，由于暂时没有收入，很容易对成员的心理产生一种慌乱的影响，于是想通过其它项目来快速获得收入来解除这种不安的心理。这种行为的一个最大问题时，它会消耗掉巨大的人员成本，拖慢项目进度，而项目进度的延期，直接造成的就是成本的增加和团队成员的信心。所以，要明确什么是我们要做的，什么是我们可以做的，什么是我们绝不能做的。在项目开始之前就要做好心理建设，预估到多长时间内是会处于一种无收入而持续投入的状态。不要捡了芝麻丢了西瓜。

 

产品混乱

没有做好产品设计，甚至连用户需求分析都没有，拍脑袋想出的功能，造成重复的返工、改造。对于技术型创业来说，如果没有好的产品人员的话，确实是一个避免不了的问题。你做的不是用户想要的，用户想要的你又想不到。
追求完美

创业意味着风险与机会，对于其中的风险，是无法百分百的全面考虑到的，总有这样或那样的突发或隐藏问题会随时蹦出来，特别是我们自以为强大的产品创意，总会在用户手中被视为无物。用户是不错的产品人员。可以以最快的速度退出你们认为用户会需要的、会喜欢的功能或产品，然后在用户的反馈中快速、频繁的修正、更改，使其越来越接近用户的期望值，这样最终产品可能会和当初规划的完全不一样，但是用户更喜爱它。而快速迭代的死敌就是追求完美。追求完美没错，错就错在要知道，完美是一种最终状态，而不是初期的形态。但凡现在所有被称得上是完美的产品（当然，这里是指的是相对完美），哪个不是经过长期的持续改进才有今天这样的完美？一降生就完美的只有“完美”这两个字而已。所以，不要怕不完美，不要怕有bug，不要怕功能不全，不要怕不能全自动，不要怕兼容性不全面，这些问题都只是时间问题，而抓住用户是当务之急。让用户看到你的态度，这样他们会对你们的不完美更加宽容一些，也会更多一些期待。面对中国的互联网环境，“快速”，是生存的法门。

啰哩叭嗦

有太多太多的人总想拥有美好，总是看到别人的美好就特别的羡慕，然后再看下自己，一声轻叹。总是觉得自己这里不如意，那里不协调，既无天分也无运气，能力平平，贵人也从未出现过，然后天天眼热着旁边人的幸福快乐。
记得很久以前看过一则短文，大意是一年轻人向一老者请教如何才能成功，老者把年轻人带到自己的住所，老者打开门时年轻人看到屋内狼藉一片，老者说：“请等一分钟”，便进屋关上了门。年轻人在门口等了一分钟。一分钟后门再次打开了，不同的是，屋内井然有序，完全看不出一丝的乱。老者说：明白了吗？
我也有这种通病，总是想要完美，总是希望能尽善尽美。殊不知，完美不是一开始就完美，只看到了完美，却没有看到形成完美的过程……
人说来也贱，他能甘于贫穷，也能接受富贵，但他就是没法接受从贫穷到富贵的这个过程，我们称之为奋斗，美其名曰实现自我价值、自我价值最大化，实际上就是一个摆脱贫穷的过程，这个贫穷，可以是指财富、智慧、学识，也可以是情感。

时下，当我洗完澡，光着膀子坐在空调房中的苹果电脑前，喝着冰镇饮料，享受着他人创造的“完美”时，我是惬意的。但我的内心确实极度茫然的，自己问自己，我会为了让别人感受到惬意而为他人创造完美吗？也许，我会，也不会。

Lion下编译Emacs

需要重新编译下Emacs，由于是在Lion下，之前的那篇Mac下编译Emacs已经不行了，再记录下。

生成configure

#获取源码
git clone git://git.savannah.gnu.org/emacs.git
./autogen.sh
./configure —with-ns
make
make install

./autogen.sh时提示automake和autoconf版本太低了。

升级autoconf和automake版本

curl -O http://mirrors.kernel.org/gnu/m4/m4-1.4.16.tar.gz
tar -xzvf m4-1.4.16.tar.gz
cd m4-1.4.13
./configure —prefix=/usr
make
sudo make install

cd ..
curl -O http://mirrors.kernel.org/gnu/autoconf/autoconf-2.68.tar.gz
tar -xzvf autoconf-2.68.tar.gz
cd autoconf-2.65
./configure —prefix=/usr
make
sudo make install
cd ..

curl -O http://mirrors.kernel.org/gnu/automake/automake-1.11.1.tar.gz
tar xzvf automake-1.11.1.tar.gz
cd automake-1.11
./configure —prefix=/usr
make
sudo make install

不管喜不喜欢，生活过的地方就是那样

让人窒息的熟悉感觉，穿越十年，丝毫不模糊。

还是那样的楼，还是那样的地，还是那样的景，可看它的心情却大不一样。让我唏嘘的不是岁月，而是对于猛回头发现诸事皆变的突然。

以前的人，还是在那，位置不同了。老师、同学、领导、路人，没有一个缺席，都在装扮着我的回忆。

校友，同学，我们何时荣归故里，我们何时举杯言欢，我们又何时形同陌路。

生活过的地方，因为有人，所以有事情，因为有事情，而产生经历，因经历而产生感情，因感情而牢记岁月。匆匆而过的只是时间，我们的心不会随时间的流逝而消逝，只会在灰蒙蒙的来路上，感受着熟悉的气息。

不管喜不喜欢，生活过的地方就是那样，总是会在你毫无防备的时候让你颤抖，因为真的忘不了。
