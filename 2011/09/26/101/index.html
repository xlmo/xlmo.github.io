以前的迷惑和现在的巩固

PHP高级特性中的一些迷惑和新知

区分抽象类和接口：
抽象类中可有实现，而接口是纯粹的模板，不能包括实现。
可实现多个接口，但只能继承一个父类。

静态方法

静态属性在所有对象中公用，常量属性只能时基本数据类型(字符串、数字），只能通过静态访问到，不需要加’$’,不可修改。

静态类中的self指的是解析上下文，而不是调用上下文。即

abstract class cls1
{
     public static function create()
     {
          return new self();
     }
}
class o1 extends cls1
{
}
o1::create();

self被解析为cls1,而cls1是抽象类，所以会报错。
这里要用延迟静态绑定(PHP 5.3+)，使用static代替self.static不只是静态方法调用标识符，在这里，static指的是被调用的类(o1)，而不是包含类(cls1)。

abstract class cls1
{
     public static function create()
     {
          return new static();
     }
}
class o1 extends cls1
{
}
o1::create();

扩展异常类的作用：
可扩展功能
可定义新的异常类型，进行特定的错误处理
异常最终没有被捕获将产生一个致命错误。
异常的作用只是报错，不承担解决错误的责任。

 

拦截器

__get( $property )

__set( $property, $value )

__isset( $property )

__unset( $property )

__call( $method, $arg_array)

 

浅复制：对象属性只复制引用。
